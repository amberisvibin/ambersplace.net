cohost tips

  • official list of third-party cohost tools: userscripts, userstyles, browser extensions, etc
  • unofficial list of useful stuff for cohost: more customization, tools, and guides
  • tag your stuff! explore others' tags!
  • don't use the "cohost global feed" tag you might see, i invented it by accident (people took a joke seriously) and it turned out to be a bad thing (causes various technical and social problems, goes against the spirit of the site)
  • be kind to others
  • you can put CSS in posts using inline styles and various HTML tags that accept a style attribute

    you can put CSS in posts using inline styles and various HTML tags that accept a style attribute

  • you can mix markdown and HTML. cohost's layout engine is a work of fucking art
  • cohost is community-supported and if you pay $5 (or more) a month for Cohost Plus, you get benefits like being able to upload 10-megabyte images. cohost doesn't automatically recompress or resize your images, which is lovely for things like grain-heavy photographs that get shrunken and murdered by JPEG recompression on other social media sites (especially bluesky and tumblr)
  • You Can Post Porno On Here. just make sure to mark it 18+ and use CWs for things that are likely to seriously bother someone.
  • check the settings pages for options including: silencing or muffling posts with given tags, silencing posts with particular CWs, or automatically expanding CW'd posts by default if you'd prefer to mainline that shit
  • you can embed all kinds of things that iframely supports by putting the URL on its own line, with an empty line above and below it. if you run into a tweet you can't embed (because it's NSFW or for other arbitrary twitter reasons), fxtwitter and vxtwitter both work
